# Commands run

- `terraform init` - Initializes a Terraform working directory, downloading necessary provider plugins and setting up the backend.
- `terraform plan` - Creates an execution plan, showing the changes that will be made to the infrastructure based on the current configuration and state.
- `terraform apply` - Applies the changes required to reach the desired state of the infrastructure as defined in the configuration files, creating or modifying resources as needed.
- `terraform destroy` - Destroys all resources managed by the current configuration, effectively tearing down the infrastructure.
